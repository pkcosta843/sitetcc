Sobre

Para que foi feito o site?

Para o trabalho de conclusão do meu curso de Técnico em Informática, desenvolvi um site abrangente de guia das cidades. Este projeto teve como objetivo principal fornecer informações relevantes e úteis sobre diferentes localidades, permitindo que os usuários descubram os lugares para poder visitar e conhecer, restaurantes, hospedagens. Através da minha experiência em desenvolvimento web e pesquisa, pude criar uma plataforma interativa e de fácil navegação, visando facilitar a vida dos moradores e turistas que desejam explorar essas cidades. O site é resultado de um trabalho árduo e dedicação para oferecer uma ferramenta valiosa para a comunidade.

Como foi feito o site?

O meu site foi criado usando as linguagens de programação HTML e CSS. Utilizei HTML para estruturar o conteúdo e definir os elementos da página, enquanto o CSS foi utilizado para estilizar e dar aquele visual maneiro. Combinando essas duas linguagens, consegui construir um site personalizado e com um design bacana. Foi muito legal colocar em prática os conhecimentos de HTML e CSS que adquiri durante o meu curso de Técnico em Informática.
